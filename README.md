## Recipy

Single Page Application created to browse different recipes and create your own

## Technologies used

-   TailwindCSS
-   Material Tailwind
-   Typescript
-   React
-   SWR
-   Husky

## API

App uses https://spoonacular.com/food-api to get the data about the recipes

## Install & run the app

### Sample .env

`REACT_APP_RECIPES_API_KEY=YOUR_API_KEY`

### Run development

```
git clone
yarn
yarn start
```
